\# 📦 Sistema de Gestión de Stock



Este proyecto es una aplicación diseñada para el control y administración de inventarios, permitiendo gestionar tanto el flujo de productos de manera organizada y eficiente como Tambien las ventas realizadas.



\## 🚀 Funcionalidades



\* \*\*Gestión de Productos:\*\* Alta, baja y modificación de artículos (nombre, código, precio, stock).



\* \*\*Control de Inventario:\*\* Actualización automática de existencias tras cada movimiento.



\* \*\*Buscador Inteligente:\*\* Filtros por nombre o categoría para encontrar productos rápidamente.



\* \*\*Alertas de Stock Bajo:\*\* Indicador visual cuando un producto está por agotarse.



\* \*\*Reportes:\*\* Vista general del valor total del inventario y de productos tanto agrupados como individuales.



\## 🛠️ Tecnologías Utilizadas



\* \*\*C++:\*\* Lenguaje principal utilizado para la lógica de programación orientada a objetos (POO).



\* \*\*Code::Blocks:\*\* Entorno de desarrollo (IDE) utilizado para la escritura, depuración y compilación del código.



\* \*\*Librería rlutil.h:\*\* Utilizada para mejorar la interfaz de usuario en la consola (colores, posicionamiento de cursor y limpieza de pantalla).



\* \*\*Persistencia de Datos:\*\* Uso de archivos binarios para almacenar la información de artículos, clientes y ventas de forma permanente.

\## ⚙️ Cómo ejecutar el proyecto



1\. Descarga o clona este repositorio en tu computadora.



2\. Abre la carpeta del proyecto y busca el archivo con extensión \*\*.cbp\*\* (este es el archivo de proyecto de Code::Blocks).



3\. Haz doble clic en él para abrirlo en el IDE.



4\. Presiona la tecla \*\*F9\*\* (o ve al menú \*\*Build > Build and Run\*\*) para compilar y ejecutar el sistema.



5\. El sistema se abrirá en una ventana de consola de comandos lista para usar.
